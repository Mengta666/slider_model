# 部署方式（只有docker）：
### 1. 下载本页的所有文件到你的服务器中
### 2. 去我的谷歌云盘下载打包好的模型与前后端调用代码
谷歌云盘下载地址：https://drive.google.com/file/d/1oJKObrPpepf6urYaYJxJOmyRZn7vE8OU/view?usp=sharing
### 3. root权限执行脚本 install_slider_model.sh
### 4. 当然，你也可以根据我写的dockerfile、docker-compose自定义安装
